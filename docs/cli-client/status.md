# Status

Query node status

**Flags:**

| Name, shorthand | Default               | Description                                                | Required |
| --------------- | --------------------- | ---------------------------------------------------------- | -------- |
| --node, -n      | tcp://localhost:26657 | \<host>:\<port> to tendermint rpc interface for this chain |          |

## Query node status

```bash
iris status
```
